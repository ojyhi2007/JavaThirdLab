## Задание 1.9 — Диапазон оценок (Student)

<details>
<summary>Подробнее об задании</summary>

### Класс Student

### Описание:
Создан класс **Student**, который описывает студента по имени и набору оценок.
Главная цель задания — гарантировать, что студенту можно задавать только оценки в диапазоне **от 2 до 5**.

### Логика решения:
- Имя студента хранится в поле типа `String`.
- Оценки хранятся в массиве `int`.
- При создании студента и при добавлении новой оценки выполняется проверка:
  если оценка меньше 2 или больше 5 — выбрасывается `IllegalArgumentException`.
- Метод `getGrades()` возвращает **копию массива**, чтобы оценки нельзя было изменить извне.
- Метод `toString()` выводит студента в формате: `Имя: [оценки]`.

### Пример:
- Создание студента **Вася** с оценками `3,4,5`.
- Попытка добавить оценку `6` приводит к ошибке.
- Попытка изменить массив, полученный из `getGrades()`, не меняет оценки внутри объекта.

</details>

## Задание 2.3 — Желтые страницы (PhoneDirectory)

<details>
<summary>Подробнее об задании</summary>

### Класс PhoneDirectory

### Описание:
Создан класс **PhoneDirectory**, который реализует простой телефонный справочник.
Справочник хранит пары **имя — телефон** и позволяет выполнять операции поиска и управления контактами.

### Логика решения:
- Для хранения данных используется структура `LinkedHashMap`.
- Каждому имени соответствует **один телефон**, и каждому телефону соответствует **одно имя**.
- Класс реализует основные операции работы со справочником:
  - добавление контакта
  - удаление контакта
  - получение телефона по имени
  - проверка существования имени
  - проверка существования телефона
  - получение количества контактов
  - получение всех имен
  - получение всех телефонов
  - получение всех пар "телефон — имя"
  - поиск имен по заданному префиксу
- Метод `toString()` выводит полный список контактов справочника.

### Пример:
Программа выполняет следующие действия:

Добавляет контакты:


89003337788 - Вася
89111111111 - Маша
89222222222 - Петя


Вывод справочника:


[89003337788 - Вася, 89111111111 - Маша, 89222222222 - Петя]


Далее программа:
- меняет телефон Васи
- удаляет Машу из справочника
- выполняет поиск телефона по имени
- выводит список всех контактов.
</details>

## Задание 3.5 — Трехмерная точка

<details>
<summary>Подробнее об задании</summary>

### Классы Point и Point3D

Описание  
Создан класс `Point`, описывающий точку на плоскости с координатами **X** и **Y**.

Создан класс `Point3D`, который **наследуется от Point** и добавляет координату **Z**.

### Логика решения

Координаты хранятся в виде чисел типа `double`.

Класс `Point3D` расширяет функциональность базового класса `Point`.

Метод `toString()` выводит координаты точки:
{X;Y} или {X;Y;Z}

</details>


## Задание 4.4 — Геометрические фигуры
<details>
<summary>Подробнее об задании</summary>

### Интерфейс Figure

Создан интерфейс `Figure`, описывающий геометрическую фигуру.

Интерфейс содержит метод:


double getArea()


### Реализованные фигуры

- Circle (круг)  
- Square (квадрат)  
- Rectangle (прямоугольник)  
- Triangle (треугольник)

### Логика решения

Каждая фигура реализует интерфейс `Figure` и содержит собственную формулу вычисления площади.

Метод `toString()` выводит описание фигуры.

</details>
